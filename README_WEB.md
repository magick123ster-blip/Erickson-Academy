# Erickson Portable Academy - Web Access Guide

본 프로그램을 외부(모바일, 다른 PC)에서 접속할 수 있게 만드는 방법입니다.

## 1. Ngrok 사용하기 (가장 간단함)
로컬에서 실행 중인 스트림릿을 외부로 터널링합니다.

1. [ngrok.com](https://ngrok.com/)에 가입하고 인증 토큰을 받습니다.
2. 터미널(또는 CMD)에서 다음 명령어를 실행합니다:
   ```bash
   ngrok http 8501
   ```
3. 화면에 나오는 `Forwarding` 주소(예: `https://abcd-123.ngrok-free.app`)를 브라우저에 입력하면 어디서든 접속 가능합니다.

## 2. 같은 Wi-Fi 환경에서 접속
같은 공유기를 사용 중이라면 IP 주소로 직접 접속할 수 있습니다.

1. CMD에서 `ipconfig`를 입력하여 `IPv4 주소`를 확인합니다 (예: `192.168.0.10`).
2. 다른 기기 브라우저에서 `http://192.168.0.10:8501`을 입력합니다.

---
**주의**: 외부에서 접속할 때도 API Key는 본인의 것을 입력해야 안전하게 작동합니다.
