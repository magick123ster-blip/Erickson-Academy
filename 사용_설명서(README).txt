==========================================================
🎓 에릭슨 전문 훈련 센터 (Erickson Portable Academy) 사용 설명서
==========================================================

본 프로그램은 세계적인 최면치료사 밀턴 에릭슨의 전략적 언어 패턴을 
실전처럼 훈련할 수 있도록 설계된 AI 기반 트레이닝 시스템입니다.

----------------------------------------------------------
1. 시작 전 준비사항
----------------------------------------------------------
- 인터넷 연결: 필수
- AI API 키: 아래 [부록]을 참고하여 발급받으세요.

----------------------------------------------------------
2. 설치 방법 (새 컴퓨터에서 최초 1회)
----------------------------------------------------------
Step 1: [처음_한번만_실행하세요(설치).bat] 실행
Step 2: 자동으로 파이썬과 필요한 프로그램들이 깔립니다.
Step 3: 창이 자동으로 닫히면 [한 번 더] 실행해 주세요. (환경 반영용)
Step 4: "모든 설치가 완료되었습니다" 메시지가 나오면 끝!

----------------------------------------------------------
3. 실행 및 훈련 방법
----------------------------------------------------------
Step 1: [에릭슨_훈련센터_실행하기.bat] 실행
Step 2: 브라우저가 열리면 왼쪽 메뉴에서 [API Key]를 입력합니다.
Step 3: 원하는 전략 테마를 고르고 [🚀 새 훈련 시작] 클릭!

----------------------------------------------------------
[부록] 왕초보를 위한 API 키 발급 가이드 (1분 소요)
----------------------------------------------------------

A. 구글(Google Gemini) API 키 받기 (추천: 가장 쉽고 빠름)
1. 구글 AI 스튜디오 접속: https://aistudio.google.com/
2. 본인의 구글 계정으로 로그인합니다.
3. 왼쪽 상단의 [Get API key] 버튼을 클릭합니다.
4. [Create API key in new project] 버튼을 누릅니다.
5. 화면에 나타난 영어와 숫자로 된 긴 코드를 복사해서 프로그램에 붙여넣으세요!

B. 엔비디아(NVIDIA) API 키 받기
1. 엔비디아 빌드 사이트 접속: https://build.nvidia.com/
2. 우측 상단 사람 아이콘을 눌러 가입/로그인합니다.
3. 목록에서 [google/gemma-2-9b-it] 등 원하는 모델을 클릭합니다.
4. [Get API Key] 버튼을 누르고 생성된 키를 복사하세요.

* 주의: API 키는 본인의 비밀번호와 같습니다. 타인에게 공유하지 마세요!

----------------------------------------------------------
4. 문제 해결 (Q&A)
----------------------------------------------------------
Q: 한글이 깨지거나 명령어를 못 찾는다고 나와요.
A: 제가 새로 드린 [설치] 및 [실행] 배치 파일을 사용 중인지 확인해 주세요. 
   최신 버전은 경로 문제를 완벽하게 해결했습니다.

Q: 브라우저 창이 안 떠요.
A: 검은 창에 적힌 주소(http://localhost:8501)를 직접 크롬에 입력해 보세요.
